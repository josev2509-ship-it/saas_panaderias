# SaaS Panaderías 🍞

Sistema web para gestión de conduces y distribución de productos de panadería para suplidores del programa de alimentación escolar (INABIE).

## Funcionalidades

* Generación automática de conduces
* Relación diaria en PDF
* Relación general en PDF
* Gestión de centros educativos
* Control de productos (pan, galletas, bizcochos, vegetales)

## Tecnologías

* Django
* Python
* ReportLab (PDF)
